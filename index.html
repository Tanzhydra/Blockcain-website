Petunjuk Penggunaan

Website ini dirancang untuk membantu memahami konsep dan simulasi berbagai teknik dalam **Blockchain Technology**. Ikuti langkah-langkah berikut:

  • Klik pada bagian Menu Bar di bagian atas untuk memilih simulasi atau fitur yang ingin dipelajari.
  • Menu Bar yang tersedia:
    1. Home: Tampilan awal Website.
    2. About Me: Profil pembuat Website.
    3. Hash: Masukkan input teks pada area Text Input, lalu hasil SHA-256 Hash akan muncul otomatis setiap kali teks diubah.
    4. Block: Anda dapat mengganti nomor blok, memberi input data, serta mengatur kecepatan mining. Tekan tombol Mine untuk memulai proses mining. Setelah menemukan hash dengan 4 angka nol di awal, nilai Nonce, Timestamp, dan Hash Block akan muncul.
    5. Block Chain: Masukkan data, tekan Mine, lalu Add Block untuk menambah blok baru. Perhatikan bahwa Previous Hash pada blok baru sama dengan hash blok sebelumya.
    6. ECC Signature: Klik Generate Key Pair untuk membuat Public Key dan Private Key. Tulis pesan dan tekan Sign Message untuk menghasilkan signature. Gunakan Verify Signature untuk memeriksa validitas signature.
    7. Konsensus: Simulasi dengan 3 user (saldo awal 100 koin). Buat transaksi, kirim, lalu tekan Mine Semua Transaksi untuk membuat blok baru. Jika data diubah secara manual dan tombol Verify ditekan, maka blok yang datanya diubah akan menjadi merah muda (invalid). Tekan tombol Consensus untuk memperbaikinya dengan mekanisme konsensus.
